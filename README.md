# Fracxtl
Fracxtl is a browser-based steganography tool that encrypts text messages with AES-256-GCM and hides the ciphertext inside a Mandelbrot fractal PNG image, with no pixel modification and no server required.

## Features

1. Encrypt messages with AES-256-GCM and embed them invisibly in a PNG
2. PBKDF2 (150k iterations) → HKDF key derivation for strong password hashing
3. HMAC-SHA256 authentication — wrong password fails instantly
4. Mandelbrot fractal parameters seeded by password — every key produces a unique image
5. Runs entirely in the browser, nothing leaves your machine

## Usage

1. Download or clone the repository
2. Open `index.html` in your browser
3. Enter your message and a password
4. Click Encrypt — save the generated PNG
5. To decrypt, upload the PNG and enter the same password

## Notes

* Security depends entirely on password strength — use a long passphrase
* Not audited by a professional cryptographer, use accordingly
* Decryption tab works with PNGs generated by CipherFract only
